# Core Modules

## tcp_scanner.py
Handles TCP port scanning using multithreading.

## udp_scanner.py
Sends UDP probes and reports open/filtered ports.

## service_detect.py
Built-in service mapping.

## banner_grabber.py
Attempts to read service banners.

## host_scanner.py
Converts CIDR ranges into host lists.

## vulnerability_checks.py
Performs non-exploitative vulnerability evaluations.

## report_writer.py
Exports scan output to JSON, CSV, HTML.
