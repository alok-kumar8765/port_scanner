# System Architecture

The scanner is divided into three main layers:

1. CORE LAYER  
    - TCP/UDP Scanning  
    - Service Detection  
    - Banner Grabbing  
    - Vulnerability Checks  
    - Reporting

2. UTILITIES LAYER  
    - Logger  
    - Helpers  
    - Config

3. PRESENTATION LAYER  
    - Tkinter GUI  
    - PyQt GUI  
    - CLI Interface
